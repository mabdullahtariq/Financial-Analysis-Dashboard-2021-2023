# 📊 Financial Performance Analysis Dashboard (2021–2023)

A financial analytics dashboard developed using Microsoft Excel and Power BI to analyze the financial performance of Pakistani listed companies from 2021 to 2023.

---

## 📌 Project Overview

This project focuses on analyzing historical financial data to identify trends in revenue, profitability, assets, and shareholder equity.

The dashboard provides interactive insights into company performance across multiple years and demonstrates practical Business Analytics and Financial Reporting skills.

---

## 🚀 Features

- Financial Performance Overview
- Multi-Year Trend Analysis
- Company Comparison
- Revenue Analysis
- Profitability Analysis
- Assets & Equity Analysis
- Interactive Filters
- KPI Dashboard
- Dynamic Charts
- Executive Reporting
